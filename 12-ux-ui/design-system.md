# Sistema de diseno

> Estado: En progreso | Ultima actualizacion: 2026-06-22
> Autor: Equipo del proyecto | Equipo: UX/UI

## Principios

- Claridad antes que decoracion.
- Formularios consistentes.
- Estados visibles: cargando, exito, error, vacio.
- Tablas filtrables para informacion operativa.
- Acciones criticas con confirmacion.

## Componentes base

| Componente | Uso |
|---|---|
| Tabla | Listados de horarios, fichas, ambientes y usuarios |
| Formulario | Crear o editar entidades |
| Modal | Confirmaciones y acciones puntuales |
| Badge | Estado de horario, alerta o documento |
| Filtro | Buscar por ficha, instructor, ambiente o fecha |
| Toast | Mensajes breves de resultado |

## Estados

- Borrador.
- En revision.
- Publicado.
- Cancelado.
- Con conflicto.
