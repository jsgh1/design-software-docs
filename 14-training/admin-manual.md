# Manual de administrador

> Estado: En progreso | Ultima actualizacion: 2026-06-22
> Autor: Equipo del proyecto | Equipo: Capacitacion / Operaciones

## Responsabilidades

- Gestionar usuarios, roles y permisos.
- Mantener parametros y catalogos.
- Revisar auditoria autorizada.
- Apoyar gestion de incidentes.
- Coordinar restauraciones o cambios criticos con el equipo tecnico.

## Tareas administrativas

| Tarea | Servicio |
|---|---|
| Crear usuario | iam-service |
| Asignar rol | iam-service |
| Actualizar parametro | reference-data-service |
| Revisar auditoria | audit-service |
| Revisar alertas | monitoring-service |

## Controles

- Usar cuentas nominales.
- Mantener minimo privilegio.
- Registrar cambios relevantes.
- No modificar datos directamente en BD.
