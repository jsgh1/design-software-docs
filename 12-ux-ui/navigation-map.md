## Navegacion principal

| Seccion | Usuarios | Proposito |
|---|---|---|
| Inicio | Todos | Resumen y accesos rapidos |
| Horarios | Coordinador, instructor, aprendiz | Consulta y gestion de programacion |
| Fichas | Coordinador | Gestion academica |
| Ambientes | Coordinador, administrador | Disponibilidad y reservas |
| Actores | Administrador | Instructores, aprendices y empresas |
| Documentos | Administrador, coordinador | Generacion y descarga |
| Monitoreo | Calidad, coordinador | KPIs, alertas y planes |
| Administracion | Admin | Usuarios, roles, parametros |
| Auditoria | Admin autorizado | Consulta de trazabilidad |

## Reglas

- Mostrar opciones segun permisos.
- Mantener busqueda por fecha, ficha, instructor y ambiente.
- Resaltar conflictos y alertas.
